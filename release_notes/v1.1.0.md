# Sage v1.1.0 Release Notes

**Release Date:** 2026-04-08  
**Version:** v1.1.0

---

## 🌟 Overview

v1.1.0 是 Sage 在 `v1.0.0` 正式版基础上的一次平台化升级版本。本版本的重点不再只是单一能力增强，而是围绕多入口接入、任务自动化、企业级部署、安全治理、工具执行环境进行了系统扩展。

相比 v1.0.0，v1.1.0 主要完成了四类升级：

1. **多入口产品化**：新增 CLI、浏览器扩展、浏览器自动化工具
2. **任务闭环增强**：支持循环任务、问卷采集、消息编辑重跑、实时进度反馈
3. **企业级能力补齐**：OAuth2、邮箱验证、认证与安全配置增强
4. **架构与工程升级**：共享 `common` 层、统一服务能力、重构文档体系

---

## 🤖 Part 1: SAgents Agent 内核更新

### 1.1 Agent 执行链路增强

本版本围绕任务规划、执行质量和上下文利用率，对 Agent 能力进行了补强：

- 新增 **PlanAgent**，支持规划阶段能力扩展
- 新增 **SelfCheckAgent**，增强结果校验与执行质量控制
- 新增 **MemoryRecallAgent**，支持记忆召回与上下文补充
- 新增 **ToolSuggestionAgent**，提升工具推荐和工具选择体验
- 新增 **用户输入优化**能力，改善复杂输入的可执行性
- 新增 **历史消息压缩**机制，降低长上下文任务的 token 压力

### 1.2 会话与消息能力升级

- 支持最后一条用户消息的**编辑与重跑**
- 增强流式消息与执行进度反馈
- 增加推理/思考类消息展示类型
- 补强 Agent 能力统计与使用数据采集
- 持续优化 Session 检查、恢复与调试体验

### 1.3 工具与工作流能力增强

- 新增 **问卷工具（Questionnaire Tool）**，支持信息采集型任务
- 文件更新工具、工具结果截断、工具调用展示进一步增强
- 支持更丰富的任务交付/执行流消息展示
- 提升复杂任务下的工具调用稳定性与可观测性

---

## 💻 Part 2: 应用层与产品能力更新

### 2.1 新增 CLI 命令行入口

v1.1.0 引入了 Sage CLI，进一步拓展了产品使用场景：

- CLI MVP 落地，支持基础对话与运行
- 支持配置管理、Doctor 检查、Session 辅助命令
- 支持 Session inspect、follow-up 等工作流
- 支持技能控制与统计上下文能力
- 补充独立 CLI 使用文档

### 2.2 浏览器扩展与浏览器自动化

- 新增 **Chrome Extension** 目录与基础能力
- 新增浏览器桥接、浏览器能力识别与浏览器工具集
- 支持浏览器自动化任务执行
- 新增 Draw.io 文件预览/渲染支持，补强浏览器侧工作流

### 2.3 工作台与聊天体验升级

桌面端/Web 端的聊天与工作台能力持续增强：

- 支持更多工作台渲染器与工具卡片
- 新增音频、视频、远程文件、Mermaid、Draw.io 等渲染能力
- 优化文件预览、代码预览、消息输入框和工具展示
- 新增任务进度消息、交付消息、推理内容展示
- 强化移动端、侧边栏、面板交互体验

---

## 🔌 Part 3: IM、外部集成与自动化更新

### 3.1 IM 渠道能力扩展

本版本对 IM 能力进行了明显增强：

- 新增 **WeChat Personal (iLink)** 支持
- 增强企业微信、飞书、钉钉消息收发能力
- 支持文件消息接收与发送
- 支持渠道自动启动、配置恢复与状态保持
- 修复重复连接、会话复用、配置冲突等稳定性问题

### 3.2 自动化与任务调度

- 新增循环任务/定时任务管理能力
- 系统设置页加入自动化配置与管理体验
- 任务调度服务和消息格式进一步完善
- 增强执行过程中的动态进度提示和状态反馈

### 3.3 统一外部能力接入

- 搜索能力升级为**统一搜索服务**
- 图片生成升级为**统一图片生成服务**
- 支持多搜索 Provider / 多图片生成 Provider 接入
- 提升外部工具体系的扩展性和工程一致性

---

## 🛡️ Part 4: 企业级基础设施与安全增强

### 4.1 认证与账号体系增强

- 新增 **OAuth2** 服务端支持
- 新增登录邮箱验证能力
- 梳理并收敛认证模式配置
- 改进登录弹窗、认证流程与兼容路径

### 4.2 安全与部署加固

- 强化 trusted proxy、鉴权与启动安全逻辑
- CORS 配置改为更可控的企业级模式
- 修复默认配置下的若干安全缺口
- 提升服务端启动、生命周期与环境兼容性

### 4.3 沙箱与执行环境升级

- 重构沙箱系统，支持本地 / passthrough / remote 多种模式
- 增强 OpenSandbox 兼容性
- 补齐 Node Runtime、sidecar 打包与执行支持
- 优化路径映射、权限白名单、镜像源与超时处理

---

## 🏗️ Part 5: 架构与工程体系升级

### 5.1 抽离 common 共享层

为提升代码复用和架构清晰度，本版本将大量共用能力抽离到 `common/`：

- 统一模型、Schema、Service、Core Client
- 服务端与桌面端共享核心业务逻辑
- 降低重复实现成本，提升后续维护效率

### 5.2 文档体系重建

- 重建中英文文档目录结构
- 新增 CLI、开发、配置、核心概念、排障、HTTP API 等专题文档
- README 与安装说明同步更新
- 补充部署、认证、活跃会话等专项文档

### 5.3 测试与构建优化

- 新增 CI 测试工作流
- 增补服务端核心测试
- 持续优化桌面端打包脚本与多平台发布流程
- 增强 Linux ARM64、macOS sidecar 等构建兼容性

---

## 🎯 业务价值

v1.1.0 让 Sage 从“可用的智能体平台”进一步升级为“具备多入口接入、自动化调度、企业级安全治理和更完整工具执行环境的智能体平台”。

这意味着：

- 用户可以通过桌面端、Web、CLI、浏览器扩展等多种方式接入 Sage
- 复杂任务可以通过问卷、定时调度、实时进度、重跑机制形成更完整闭环
- 企业部署可以获得更成熟的认证、安全与配置能力
- 开发者可以基于统一的共享架构更高效地扩展系统能力
