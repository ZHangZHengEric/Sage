# 小红书图文爆款写作指导

## 平台特点
- 小红书用户以18-35岁年轻女性为主，核心用户群体是都市白领、学生党，注重生活品质、美妆、时尚、美食、旅行等内容
- 平台强调"种草"文化，用户喜欢真实、亲切的个人分享
- 图片质量要求极高，需要高清、美观、有氛围感，通常采用9宫格或单张大图形式
- 内容风格偏向个人化、生活化、情感化，强调真实感和亲和力
- 爆款内容通常具有实用性、创新性、情感共鸣或视觉冲击力
- 小红书的算法偏好原创、高质量、互动性强的内容

## 爆款标题技巧
1. **数字吸引**：例如"5分钟搞定的宿舍早餐食谱"、"3个步骤让皮肤变光滑"、"2024年必入的5款平价护肤品"
2. **问题引导**：例如"夏天出油怎么办？试试这个方法"、"学生党预算有限如何护肤？"、"职场新人如何快速融入团队？"
3. **结果承诺**：例如"坚持一周，腰围减少2cm"、"用了这个方法，再也没长过痘痘"、"这个技巧让我每天多睡30分钟"
4. **好奇心激发**：例如"我尝试了这个冷门方法，结果震惊了"、"被问爆了的包包链接，终于找到了"、"这个隐藏功能99%的人都不知道"
5. **情感共鸣**：例如"打工人必看，这个方法让我不再加班"、"社恐福音，这样做再也不怕社交了"、"异地恋必看，维持感情的3个小技巧"

## 内容结构
1. **封面图**：选择最吸引人的图片作为封面，通常是效果对比图、成品图或氛围感照片
2. **引言**：用亲切的口语化表达介绍背景，引起读者共鸣，例如"最近发现了一个超好用的方法，必须分享给你们！"
3. **正文**：分点说明，每点配一张相关图片，结构清晰，重点突出
4. **总结**：总结关键点，强调效果或价值
5. **互动**：鼓励读者评论、点赞、收藏，例如"你们有什么好用的方法也可以分享在评论区哦～"
6. **标签**：添加5-8个相关话题标签，包括大话题和小话题，增加曝光

## 真人感技巧
1. **口语化表达**：使用小红书特有的口语化表达，例如"家人们"、"宝子们"、"谁懂啊"、"绝了"、"yyds"
2. **个人经历**：分享自己的真实故事和感受，例如"我之前一直被痘痘困扰，试了很多方法都没用，直到遇到了这个..."
3. **适当的错别字**：使用符合小红书风格的错别字，例如"炒鸡"、"好ci"、"灰常"、"绝绝子"、"yyds"
4. **表情符号**：大量使用符合平台风格的表情符号，例如🤩、😍、✨、🔥、💯、🌟
5. **真实图片**：使用自己拍摄的图片，避免过度修图，保留一些真实的细节，例如"刚拍的，没修图，真实效果就是这样"
6. **细节描述**：加入具体的使用感受和细节，例如"质地像酸奶一样，涂在脸上凉凉的，很舒服"

## 爆款案例分析
- **案例1**："这个平价护肤品让我摆脱了痘痘肌，附使用前后对比"
  - 特点：真实对比图、个人经历分享、具体使用方法、效果承诺

- **案例2**："宿舍党必备的5款神器，最后一个绝了"
  - 特点：针对性强、数字吸引、好奇心激发、实用性高
  - 效果：高点击率、大量转发、被多个账号转载、引发购买热潮

#### 渲染图片

使用Python脚本渲染图片：

```bash
# 1) 默认主题 + 手动分隔分页
python scripts/render_xhs.py content.md -m separator -o output

# 2) 固定 1080x1440，自动缩放文字
python scripts/render_xhs.py content.md -m auto-fit -o output

# 3) 自动切分分页（推荐：内容长短不稳定）
python scripts/render_xhs.py content.md -m auto-split -o output

# 4) 动态高度（允许不同高度卡片）
python scripts/render_xhs.py content.md -m dynamic --max-height 4320 -o output

# 5) 切换主题
python scripts/render_xhs.py content.md -t playful-geometric -m auto-split -o output
```

#### 可用主题

- `default`：默认简约浅灰渐变背景
- `playful-geometric`：活泼几何（Memphis）
- `neo-brutalism`：新粗野主义
- `botanical`：植物园自然
- `professional`：专业商务
- `retro`：复古怀旧
- `terminal`：终端命令行
- `sketch`：手绘素描

#### 分页模式

- `separator`：按 `---` 分隔符分页
- `auto-fit`：固定尺寸下自动缩放文字
- `auto-split`：按渲染后高度自动切分分页
- `dynamic`：根据内容动态调整图片高度

#### 图片规格

- 尺寸比例：3:4（小红书推荐比例）
- 基准尺寸：1080×1440px
- 包含：封面（cover.png）和正文卡片（card_1.png, card_2.png, ...）

## 写作注意事项
1. **保持真实性**：避免虚假宣传，真实分享体验，标注"无广"增加可信度
2. **注重实用性**：提供具体的方法和建议，让读者看完就能上手
3. **优化图片**：确保图片清晰、美观、有氛围感，统一色调和风格
4. **互动引导**：鼓励读者评论、点赞、收藏，及时回复评论
5. **标签策略**：合理使用话题标签，包括大话题（如#护肤）和小话题（如#痘痘肌护理）
6. **发布时间**：选择用户活跃时间发布，通常是晚上8-10点
7. **封面优化**：封面图要吸引人，有视觉冲击力，能快速传达内容核心
8. **标题长度**：标题控制在20-25字之间，既包含关键词又有吸引力
